# URI

Estúdio de identidade digital para psicólogos.
Landing pages, sites institucionais e campanhas Google Ads.

**Site:** [uriads.com](https://uriads.com)
**Contato:** contatouriads@gmail.com
**Localização:** Lisboa · Rio de Janeiro

---

## Sobre

URI — do hebraico *אוּר* (ur), "luz" — é um estúdio dedicado a iluminar a prática de psicólogos no ambiente digital. Trabalhamos dentro do que a OPP e o CFP permitem, porque marketing sério respeita a profissão que serve.

## Stack

Site construído em HTML, CSS e JavaScript puro. Uma única página, uma única interação — cursor acende a tela.

- Tipografia: Instrument Serif + JetBrains Mono (Google Fonts)
- Sem frameworks, sem bundler, sem build step
- Deploy automático via GitHub Pages
